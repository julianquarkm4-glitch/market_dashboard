EVC Terminal — Data API

This repo auto-generates market data for the EVC Terminal dashboard.

Updated Mon-Fri at 16:30 ET via GitHub Actions.

Data Files

  • screener.json — Momentum scan results + quant metrics
  • sectors.json — Sector / industry / country ETF tracking
  • breadth.json — Market breadth indicators + regime
  • calendar.json — Upcoming economic events + earnings
  • meta.json — Build metadata
